# researcher

**وكيل باحث موثّق داخل Claude Code**
*A cited research agent for Claude Code*

اطلب موضوعاً، واحصل على تقرير: يبحث في الإنترنت من زوايا متعددة، يفتح المصادر
ويقرأها، يتحقّق من كل معلومة من أكثر من مصدر، ثم يسلّم خلاصة موثّقة بمراجع، ويميّز
المؤكّد من الاستدلالي، ويذكر ما لم يجده بدل أن يخترعه.

```
استخدم وكيل researcher وابحث لي عن أنظمة كشف وصدّ المسيّرات المتاحة تجارياً
```

---

## التثبيت · Install

```bash
git clone https://github.com/hussamalraddadi/researcher.git
cd researcher
bash install.sh
```

المثبّت ينسخ ملفاً واحداً إلى `~/.claude/agents/`. لا يحمّل شيئاً، ولا يحتاج مفتاحاً.

أعد تشغيل Claude Code، ثم اطلب من كلود أن يستخدم الوكيل.

> **لا يحتاج مفتاح API.** الوكيل يستخدم بحث الإنترنت المدمج في Claude Code.

### الإزالة · Uninstall

```bash
bash install.sh --uninstall
```

## كيف تستخدمه · How to use

قل لـ Claude Code شيئاً مثل:

- «استخدم وكيل **researcher** وابحث لي عن …»
- «شغّل الباحث ويجمع لي مصادر عن …»

أمثلة على ما يصلح له:

| النوع | مثال |
|---|---|
| **مسح سوق** | ما الشركات التي تقدّم أنظمة كشف وصدّ المسيّرات، وما مميزات كل نظام؟ |
| **مراجعة موضوع** | ما أحدث الممارسات في حماية البنية التحتية من الطائرات المسيّرة؟ |
| **تحقّق من معلومة** | هل الرقم الفلاني صحيح؟ من أين مصدره؟ |
| **مقارنة خيارات** | قارن بين ثلاثة حلول من حيث المدى، التقنية، والتكلفة المنشورة. |

## ماذا يضمن لك · What it guarantees

- **كل رقم له مصدر.** لا رقم من الخيال.
- **تحقّق من مصدرين** لكل معلومة جوهرية قبل اعتمادها.
- **تمييز صريح** بين `[مؤكّد]` و`[استدلالي]` و`[غير معروف]`.
- **قسم مصادر** في نهاية كل تقرير، وقسم **حدود البحث** يذكر ما لم يُوجد وما تعارض.
- **يبحث في المصادر المفتوحة والمشروعة فقط.** الأبحاث الدفاعية والأمنية (مثل صدّ المسيّرات وحماية المنشآت) مشروعة ما دامت من مصادر منشورة علناً. لا يساعد في بناء سلاح أو التفاف على قانون.
